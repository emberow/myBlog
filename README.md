# quick start 

## use docker-compose ( frontend、backend、postgres )
``` bash
docker-compose up
```

### run frontend
```
# use node 21
$ cd frontend
$ npm i
$ npm run dev:run
# run frontend at http:localhost:8000/
```

### run backend
```
# use node 21
$ cd backend
$ npm i
$ npm run build
$ npm run start
# run backend at http:localhost:3000/
```
